# antara1005.github.io

Personal portfolio site for **Antara Banerjee** — Robotics & Controls researcher (PhD, ECE, University of Washington).

🔗 **Live:** https://antara1005.github.io

## About

A single-page static portfolio covering research experience, technical skills,
publications, and awards. No build step or dependencies — just plain HTML and CSS,
served via GitHub Pages.

## Structure

| File            | Purpose                                  |
| --------------- | ---------------------------------------- |
| `index.html`    | The entire site (markup + inline styles) |
| `portrait.jpeg` | Hero portrait image                      |

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Editing & deploying

Edit `index.html`, then push to `main` — GitHub Pages rebuilds automatically:

```bash
git add -A
git commit -m "Update site"
git push
```
